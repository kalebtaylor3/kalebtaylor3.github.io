Evermania

Description

  • Independent Group Project

  • Role: Team Lead

  • Release date: Summer 2023

Project

Evermania was a released mobile project built by a two-person team to explore production mobile development, live deployment concerns, and monetization features. From a technical standpoint, it combined an infinite content loop, in-game economy features, ad integration, and the kind of reliability work needed to ship on mobile platforms rather than just prototype for desktop.

Technical Focus

Key technical areas included procedural or infinite content flow, responsive input and movement, cosmetic unlock systems, ad and reward integration, and the practical constraints of getting a mobile build ready for release. It was a useful project for learning how user-facing features, monetization, and platform requirements all interact.

My Work

As lead programmer, I worked across the full technical stack of the project in Unity and C#. I implemented the infinite grid system, movement logic, cosmetic shop features, and playable ad integrations while helping the project stay stable enough for mobile release. This project gave me hands-on experience building software features that needed to be both technically sound and practical for a real shipping product.
