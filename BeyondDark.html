Project

Beyond Dark was a four-month capstone project developed by an eight-person team and showcased at the 2023 Level Up event. From a software perspective, it was a collaborative production project that required coordinated systems development, rapid iteration, and shipping under a hard deadline.

Awards:
Level Up winner for Best Artistic Achievement presented by @gameloftto 2023.
Level Up winner for Best Technical Innovation presented by @ukengames 2023.

Technical Focus

The project combined AI behaviour, enemy systems, player interaction logic, and real-time gameplay flow inside a shared production codebase. It was a strong example of building systems that had to be reliable, readable, and easy for the rest of the team to integrate with while the project scope continued to evolve.

My Work

As programming lead, I owned the implementation and integration of core gameplay systems in Unity using C# and HLSL. My work included building and stabilizing enemy behaviours, interaction systems, and moment-to-moment gameplay logic while helping the team keep features aligned under a fixed deadline. This project strengthened my ability to lead technical work, work across disciplines, and keep a shared codebase moving when multiple systems depend on each other.
