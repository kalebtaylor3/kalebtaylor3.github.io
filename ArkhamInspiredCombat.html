Project

This project is a C++ combat framework built in Unreal Engine 5 to explore target selection, state coordination, timing-sensitive input, and multi-agent behaviour. While the presentation is inspired by the Arkham series, the real purpose of the project was to design and implement a clean technical system from scratch and prove out the architecture in a fast-paced real-time scenario.

All visual assets are licensed from the Fab Marketplace and Humble Bundle.

My Work

As the sole developer, I designed and implemented the entire framework in C++. I built a targeting and chaining system that chooses the best follow-up target, rotates the player cleanly into the next action, and keeps combat readable under rapid state changes. Combat timing is animation-driven, with montages and notifies controlling hit windows, collisions, damage events, and feedback cues. I also implemented counter and parry flows with narrow timing windows, plus an AI coordination layer that uses Behaviour Trees and a lightweight group manager so multiple enemies can share the encounter without creating noise or inconsistent behaviour. The result is a good example of state management, event timing, and multi-actor coordination in a real-time system.
